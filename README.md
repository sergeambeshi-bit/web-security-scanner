# Web Security Scanner

Python-based tool to scan websites for common vulnerabilities, HTTPS issues, and missing security headers. Includes OWASP top 10 checks and generates a simple security report.

## Features
- HTTPS verification
- Security headers check
- OWASP top 10 vulnerability check
- Generates security report

## Tech Stack
- Python 3
- Requests & BeautifulSoup

## Usage
```bash
git clone https://github.com/sergeambeshi-bit/web-security-scanner.git
cd web-security-scanner
pip install -r requirements.txt
python scanner.py
